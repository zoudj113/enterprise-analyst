# Enterprise Analyst Skill

## 运行准则

- **严谨性**：所有财务数据必须直接引用自源文件，严禁幻觉。若文件中缺失数据，须明确标出。
- **对比思维**：如果同时处理多个财报文件，应自动生成对比矩阵。
- **结构化输出**：始终以清晰的 Markdown 标题和表格形式呈现结果。

## 分析逻辑流程

### 第一步：业务拆解

分析企业”靠什么赚钱”。区分核心业务与边缘业务，观察各板块营收占比及同比增速。
- **利润含金量**：判断企业赚的是否是真金白银（经营性现金流）而非应收账款——对比净利润与经营性现金流净额，若长期经营性现金流低于净利润，说明利润质量存疑。

### 第二步：财务体检

重点关注以下指标的趋势：

- **毛利率**：体现产品竞争力和成本控制力。
- **费用率**：观察销售、管理、研发费用的变动是否合理。
- **现金流**：确认经营性现金流净额是否与净利润匹配（含金量）。

### 第三步：战略研判（护城河）

根据文件中描述的市场地位、技术壁垒、特许权或用户粘性，总结其护城河深度。
- **盈利可持续性**：判断企业赚钱是否可持续——考察客户复购率、行业需求稳定性、竞争格局变化趋势，高复购率+稳定需求+寡头格局意味着盈利可持续性强。
- **资本投入依赖度**：判断企业赚钱是否需要大的资本投入——对比资本支出占经营性现金流的比重，轻资产模式（低 CAPEX/OCF）优于重资产模式（高 CAPEX/OCF），后者需要持续烧钱维持增长。

### 第四步：股权架构与治理分析

拆解企业的股权控制关系与公司治理结构，重点关注：

- **控制权结构**：绘制控股股东→中间持股平台→上市公司的控制链，计算实际控制人最终持股比例及表决权。注意区分直接持股、间接持股和一致行动安排。
- **股本演变**：梳理公司从设立到上市的关键股本变动节点（发起设立、增资扩股、股权转让、员工激励、配售/定增、回购），识别是否存在股权纠纷或历史瑕疵。
- **股东回报记录**：分析分红支付率、累计分红金额及股份回购记录，判断管理层对股东回报的重视程度和持续性。
- **子公司与全球布局**：列出主要境内外子公司、持股比例及功能定位，评估跨境架构的税务效率与合规风险。
- **治理架构**：审查董事会构成（独立非执行董事占比）、专门委员会设置、上市规则豁免情形，评估治理独立性和规范程度。
- **股权激励**：梳理存量股权激励计划的授予进度、归属状态及潜在摊薄影响。
- **股权架构风险**：扫描控股股东控制权过大、少数股东保护不足、跨境架构复杂度过高等潜在问题。

## 期望输出格式

1. **企业概况**（需包含以下要素）：
   - 公司全称、上市地及股票代码。
   - 核心业务模式（靠什么赚钱，轻资产 vs 重资产）。
   - 行业地位与市场份额（是否为龙头，主要竞争对手）。
   - 当前规模数据（门店数、用户数、覆盖区域等，视行业而定）。
   - 近年发展脉络（关键收购、战略转型、里程碑事件）。
   - **股权架构**：控制权结构图（控股股东→上市公司控制链）、股本演变关键节点、股东回报记录（分红支付率/累计分红/回购）、境内外子公司布局表、治理架构（董事会/专门委员会/上市豁免）、股权激励存量及摊薄影响、股权架构风险扫描。
2. **核心指标表**（包含当前数值与同比/环比变化，必须涵盖以下指标）：
   - 营业收入及增速。
   - **毛利润及毛利率**（若源文件未直接披露，须根据营收减去营业成本推算并注明）。
   - 经营利润及经营利润率。
   - 归母净利润及净利润率。
   - 经调整 EBITDA 及 EBITDA 率（若披露）。
   - 经营性现金流净额。
   - 费用率拆解（销售/管理/研发费用占收入比）。
3. **竞争力分析**（护城河总结）。
4. **风险预警点**（清单形式）。

### 第五步：风险扫描

识别以下红旗（Red Flags）：

- 坏账准备异常。
- 核心业务增速大幅放缓。
- 行业政策变动或替代品威胁。
- 控股股东控制权过大或股权纠纷。
- 跨境架构复杂度过高带来的合规风险。
- 股权激励计划的潜在摊薄冲击。

## 第六步：输出报告

分析完成后，将完整的分析报告保存为 Markdown 文件，输出到当前工作目录下。

- **文件命名规则**：`{公司名称}_{报告类型}_分析报告.md`（如 `华住集团_2025年报分析报告.md`）。
- **文件内容**：包含完整的分析结果，涵盖企业概况（含股权架构）、核心财务指标、经营数据分析、护城河研判、风险预警和总结六大章节。
- **表格要求**：所有财务数据以 Markdown 表格呈现，包含同比/环比变化列。
- **徽标要求**：所有评级使用星级（★）和颜色标识（🔴🟡🟢）表示严重程度。
